2. 数字(num)を与え偶数であれば「(num) is even.」、奇数であれば「(num) is odd.」と表示する関数「even_or_odd」をつくる。

出力 : 

                        
->even_or_odd(114)
114 is even.
->even_or_odd(51)
51 is odd.
                        
                    

                        
def even_or_odd(num):
    if num % 2 == 0:
        print(str(num) + ' is even.')
    else:
        print(str(num) + ' is odd.')

または

                        
def even_or_odd(num):
    append = ""
    if num % 2 == 0:
        append = ' is even.'
    else:
        append = ' is odd.'

    print(str(num) + append)

3. 数字(num)を与え正の数であれば「(num) is positive.」、負の数であれば「(num) is negative.」、ゼロであれば「(num) is zero.」と表示する関数「more_or_less」をつくる。

出力 : 

                        
->more_or_less(114)
114 is positive.
->more_or_less(-514)
-514 is negative.
->more_or_less(0)
0 is zero.
                        
                    

                        
def more_or_less(num):
    if num > 0:
        print(str(num) + ' is positive.')
    elif num < 0:
        print(str(num) + ' is negative.')
    else:
        print(str(num) + ' is zero.')

def more_or_less(num):

    elif num < 0:
        append = ' is negative.'

        append = ' is zero.'

    print(str(num) + append)
